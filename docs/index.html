What it does

Beyond installing the tools — it makes them work together on your projects.

🪄

First-run wizard

Detects installed tools, installs the missing ones, and applies config (Context7 MCP + the in-tool project-init command) to every selected tool — new and already-installed.

📄

Cross-tool context

Writes a single AGENTS.md (the cross-tool standard) plus pointers, with auto-detected real build/test/lint commands. Every tool reads the same source of truth.

🎓

Expert discovery

A bundled vetted catalog + live trusted sources + on-demand generation. Best-fit skills/agents rendered to each tool's native format — Claude .md, Codex .toml, Cursor rules, and more.

🔌

MCP propagation

Writes Context7 (and other MCP servers) into each tool's native config with env-interpolation — your API key never touches disk, only the environment.

🔁

Session continuity

Stop a session, switch tools, come back tomorrow. catchup / wrapup keep a STATE.md log (a Stop hook snapshots git in Claude Code).

💻

Cross-platform

Windows (winget), Linux (apt/dnf/pacman/zypper/apk), macOS (Homebrew). Zero runtime dependencies — just Node.js ≥ 16.

Quick start

Three ways in. No account needed to set up — you authenticate each tool with your own.

  1. One-shot (recommended): run npx ai-coding-stack for the interactive wizard, or clone & run ./setup.sh.
    git clone https://github.com/alexxanderdiaz/ai-coding-stack.git && cd ai-coding-stack && ./setup.sh
  2. Non-interactive: node setup.js --tools all sets up all eight tools in one go.
  3. Scaffold a project: in any tool, say project-init (or run node project-init.js . --about "what it is") — writes AGENTS.md + configures the experts that project needs.
